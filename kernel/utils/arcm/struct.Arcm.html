    arc: Arc<RawMutex<T, false>>,

Expand description

Wrap the given type into an Arc and a Mutex. Arc allow multiple reference on the same data between threads Mutex allow any type to be Send and ensure safe access to the underlying data

Clone the current Arc and send it to the function pass in paramters

impl<T, U> CoerceUnsized<Arcm<U>> for Arcm<T>where T: Unsize<U> + ?Sized, U: ?Sized,

Allow type coercion from Arcm<T> to Arcm<U> where U is Unsized and coercion exist from T to U This is used to store dyn Trait from a struct that implement the trait for example

Examples

trait Foo {
    fn bar(&self) -> u32;
}
struct Foobar {}
impl Foo for Foobar {
    fn bar(&self) -> u32 { 42 }
}
fn exmaple() {
    let foo: Foobar = Foobar{};
    let arcm: Arcm<dyn Foo> = Arcm::new(foo);
    assert_eq!(arcm.lock().bar(), 42);
}
